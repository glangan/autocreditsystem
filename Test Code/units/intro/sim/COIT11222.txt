This course is designed for students who have had little or no programming experience. The course aims to teach students principles, design and development of object-oriented programs. It covers topics such as modern IDEs, pseudocode, variables, constants, data types, operators, expressions, statements, classes, objects, inheritance, loops, methods, passing parameters and arrays. Students will learn how to design, implement and test programs using a modern IDE.
Introduction to Object Technology and Java
